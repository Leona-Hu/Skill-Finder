Build & Release EngineerLocation:  New York, NYHave the opportunity to join a Software Product firm located in New York City as a Build & Release Engineer performing automation of the application lifecycle management (ALM) process. In this role you will lead the design, implementation, and support of the build and deployment solutions for the firm’s software. You will work closely with software platform and application teams to create new code deployment workflows, automate processes, and improve source c& version control and configuration management systems. You will need to influence decisions and priorities regarding the creation and maintenance of such systems, including Continuous Build and Continuous Integration systems and establishment of best practice development standards and guidelines. You'll need to be collaborative and hands-on, someone who likes to work in the trenches and can lead strategic projects to improve the firm’s SDLC.Required Skills:• Must have 4+ years of experience in Application Lifecycle Management (ALM), Source & Version Control and Configuration Management using GIT• Experience with Build Automation and Release Management, as well as tools such as Jenkins, Bamboo, Travis, or similar.• Demonstrated experience implementing workflows for multiple parallel releases using branching, tagging, etc.• 2+ years of experience programming in Python• A solid understanding of source control concepts and workflows, source control administration, distributed version control, and the logistics of managing large code bases with many contributors• A strong understanding of package concepts• Experience in C/C++ programming, compiling, and linkingDesired Skills:• Knowledge of Debian package manager• Familiarity with GitHub API• Familiarity with build tools such as SCons or Waf
